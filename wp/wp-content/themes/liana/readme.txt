liana - Free Responsive WordPress Blog Theme

== Description ==
Liana is a super clean & elegant Responsive WordPress Blog Theme. It's perfect for your personal, travel, lifestyle, food & recipes, storytelling  Blog. If you want to focus or showcase your content in a timeless manner, liana is right choice for you. It follows the latest trendy design with a combination of pure elegance.
Liana Pro has more prominent design, 5 custom widget, extensive theme options with fully documentation.Find out Liana Pro here, https://shop.jouleslabs.com/lianapro

* global is removed from functions.php file line no 93.
